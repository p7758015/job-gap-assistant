# Job Gap Assistant - AI-анализ соответствия кандидата вакансии (LLM + FastAPI)

Job Gap Assistant - AI-сервис для анализа соответствия между требованиями вакансии и навыками кандидата.

Позволяет автоматически выявлять пробелы (skill gaps), оценивать уровень соответствия и формировать персональный план развития.

Подходит для карьерных сервисов, HR-платформ, образовательных продуктов и специалистов, готовящихся к собеседованиям.

⚡ **Важно:** проект реализован в формате *vibe coding* с использованием **Cursor (AI-IDE)** - архитектура, API и логика были сгенерированы и доработаны через AI.

---

## Problem

Соискатели и разработчики сталкиваются с типовыми проблемами:

* непонятно, насколько они соответствуют вакансии
* сложно объективно оценить свои навыки
* нет четкого понимания, что учить дальше
* подготовка к собеседованиям занимает много времени

Вакансии описывают требования, но не дают персонального фидбэка.

---

## Solution

AI-сервис анализирует вакансию и профиль кандидата и:

* извлекает ключевые требования (must-have / nice-to-have)
* оценивает уровень соответствия (fit score)
* объясняет результат
* выявляет пробелы в навыках
* предлагает проекты для прокачки
* формирует roadmap развития на 2–4 недели

Позволяет быстро понять, «где ты сейчас» и «что делать дальше».

---

## How it works

1. Пользователь передает текст вакансии
2. (Опционально) добавляет профиль кандидата
3. Система:

   * анализирует требования вакансии через LLM
   * сопоставляет с профилем кандидата
   * выявляет gaps
   * формирует рекомендации и roadmap
4. Возвращает структурированный JSON-ответ

## API

### POST /analyze

Анализ вакансии и профиля кандидата

**Пример запроса:**

```json
{
  "job_description": "Python backend engineer with FastAPI and PostgreSQL",
  "candidate_profile": "2 years Python, FastAPI, basic PostgreSQL"
}
```

**Пример ответа:**

```json
{
  "skills": {
    "must_have": ["Python", "FastAPI", "PostgreSQL"],
    "nice_to_have": ["Testing"],
    "bonus": ["System design"]
  },
  "fit_score": 78,
  "gaps": ["Advanced PostgreSQL"],
  "projects": [
    {
      "title": "FastAPI + PostgreSQL проект",
      "estimated_duration_weeks": 2
    }
  ],
  "roadmap": [
    {
      "week": 1,
      "focus": "Тестирование API"
    }
  ]
}
```

Ответ валидируется через Pydantic-схему.
